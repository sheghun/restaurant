# Restaurant

This is a repo that contains two mono repo, one for the frontend and one for the backend

## To Build And Run

cd into the mono-repos and install their dependencies.

```bash
    $ cd front-end; yarn install 
```

```bash
    $ cd back-end; yarn install
```

```bash
    $ yarn build
```

```bash
    $ docker-compose up
```
